# FlyToSky : 

FlyToSky是一款内核为sing-box的vpn代理软件,

支持的协议:
  vless协议,vmess协议,tuic协议,Shadowsocks协议.支持ECH

  节点设置 优选ip 功能,直接把 单一节点 一键变成 集群组

  ![Image text](https://github.com/sky336699/flytosky/blob/main/YouXuanIp.png)

  ![Image text](https://github.com/sky336699/flytosky/blob/main/FlyToSky.png)

崩溃日志选用sentry.io,数据存于美国,并且sentry.io的dns解析与日志流量均走本地的代理流量,无流量特征.
